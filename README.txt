This is a chat app made in C, can create a chatroom some number of clients.

To run:
Run server first
    ./server/SERVER
    (optional command line arg for IP and port can be given, default is 127.0.0.1, 8096)
Next, run clients. (In a new terminal, or a different computer on the same network)
    ./client/client
    (command line arg for server's IP and port need to be given, but default is 127.0.0.1, 8096)
